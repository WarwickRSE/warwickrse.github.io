Detailed Description

Counterpart to springTiePrimary.

A simple fixed force to be applied. To ensure conservation, this should be the equal-and-opposite force from some other entity to allow connections between them. This end is assumed to be fixed to a specific ball in the ballset (this could be changed if needed).

Caveat:
The force here is simply applied by value, so must have already been populated. In Debug mode this is strictly enforced.
