Detailed Description

template<typename integrator_T>
class KCP< integrator_T >

Class for a pair of Kinetochore items forming a linked pair, a dumbell of 2 balls joined by a spring. Has position, velocity and pair separation (at equillibirum) as well as storing (for convenience) the 3-d orientation in alt-azimuth terms. Dynamics of the pair members are done by using a ballset of size 2 and the given integrator, other features are handled by the Kinetochore individual members

Copy constructor

For simplicity, KCs are not copyable, and KCPairs are only copied during setup, since they cannot be created or destroyed, and pointers into them are used elsewhere in the code. This instead default initialises the pair and KCs, and copies the force parameters from the other pair - i.e. a "fresh" setup with the same params.
