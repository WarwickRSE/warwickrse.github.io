Class Simulation
Simulation SETUP will establish the Spindles and Kinetochores and AFTER SETUP COMPLETES these will never be deleted. It is therefore safe to refer into them using pointers/references elsewhere in the code. DO NOT write code which modifies the SP or KC vectors after setup completes. DO NOT write code which refers to Microtubules this way! They can be and are regularly added and destroyed, and thus cannot be assumed stationary in memory.
