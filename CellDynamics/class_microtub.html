Detailed Description

template<typename integrator_T, typename integrator_Tip>
class Microtub< integrator_T, integrator_Tip >

A microtubule.

Microtubule class, containing set of balls to handle dynamics (using forces as prescribed by given integrator), and also logic for polymerising, and other effects Parameters set by the input system are included via the MTInternals struct. Note that tubules have a virtual extra "tip" ball to handle polymerisation/depolymerisation
