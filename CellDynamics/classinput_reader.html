inputs inputReader::read ( std::string  filename)
inline

Read inputs Reads entire contents of given filename (assumed to be full path or relative to current working directory) and parses values into inputs object. Will ignore anything following a ';' on any line and assumes key = value syntax. Any line without an '=' will just be ignored. Any unknown keyword will be ignored with a warning. Any missing items will use default values specified by inputs class. If file is missing or unreadable, all values will be defaults.
